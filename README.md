Compilation with latexmk
------------------------

Compile everything:

    latexmk

Clean up everything:

    latexmk -C


Compilation with rubber
-----------------------

Compile everything:

    rubber -Wall dda-*.tex

Clean up everything:

    rubber --clean dda-*.tex
